# AgentOffer RFCs

The proposal process for changing AgentOffer Protocol.

Use this repository for breaking changes, governance updates, and major design proposals.

## When To Open An RFC

- New protocol fields or objects
- Renaming or removing existing fields
- Contract changes that affect compatibility
- Governance or contribution process changes

## When A Direct PR Is Enough

- Typo fixes
- Clarity improvements that do not change semantics
- Small editorial improvements

## Start Here

Use `templates/rfc-template.md` when opening a new proposal.
